            <div class="text-long"><div><div class="photoswipe-gallery"><p>Learn how to use Akana to create an app, call an API, and request access to test <a href="https://www.perforce.com/resources/aka/api-security-best-practices">API security</a>.</p><p>Here is your step-by-step process.&nbsp;</p><p>As an API consumer, if you want to call an API that has security attached, you first have to create an app.</p><ol><li>Click “apps” from the top nav, then “add app” from the drop down. Then give the app a name, summary, and description.</li><li>Next, you scroll down to enter the app’s version ID, as well as version notes. You can either add the app ID and shared secret manually, or allow the system to generate them. You can also set the visibility to public, private, or registered users.</li><li>Finally, scroll down to the lower right hand corner and click “finish.” Now your app has been created.</li><li>If you wish to call an API with security, click “APIs” from the top nav and click “all APIs” from the drop down menu. Find the API you wish to add security to from the list, or use the search functionality. Click “access” in the upper right hand corner.</li><li>Next, you select “my test app.” Click “next.” Then select the environment you want to test in and any licenses you wish to enable.</li><li>The administrator would then get an access request. They could review all associated app information and either approve or reject the request. The app is then contracted to be able to call the associated API.</li><li>If you with to invoke the API, navigate to the documentation page found on the left hand navigation. Next, click “setup” to tell the test client what app you’re going to use to call this API. Select “my test app” which now appears in the list of contracted apps. Then click “save.”</li><li>If you want to call this GET account operation, it doesn’t require any parameters. You’ll notice the invoke is disabled. Since this API has an <a href="https://www.perforce.com/blog/aka/what-is-oauth">OAuth security</a> policy attached, you must click the “security” tab and click “get token.” The OAuth provider will issue an OAuth access token with the read scope. Click “authorize” to display your access token. Now the client has an access token to be able to call this API.</li><li>Finally, click “invoke” and the API is called. &nbsp;</li></ol><p>Reach out to find out if you qualify for a free 6-month trial of Akana.</p><p><a href="https://www.perforce.com/products/akana/free-trial">Free Trial</a></p></div></div></div>
